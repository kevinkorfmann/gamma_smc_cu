# Archived postprocess artifacts (2026-04-11 rerun)

These files were produced by the 2026-04-11 genome-wide rerun and are
superseded by the v4.1 manuscript data assets. They are kept here for
historical reference and are not referenced by any v4.1 figure, table,
test, or verify script.

Files still live in `analysis/genome_wide/postprocess/` because they
are actively used by v4.1:

- `genes_sd_flag.csv` — SD-mask flag, used by verify/ and tables/
- `pathway_enrichment.csv` — used by tests/test_claims.py
- `top50_geom_mean_no_sd.csv` — used by tables/gen_tables.py
